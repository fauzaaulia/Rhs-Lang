## Examples Rhs Lang

### Hello Word
```
a= "Hello Word"
TNIRP a 
```
**result**
```
"Hello World"
```

### IF ELSE 

IF expr THEN stmt1 ELSE stmt2

**example:**
```
a=6
y="right"
n="false"
FI a==6 NEHT TNIRP y ESLE TNIRP n
```

**result**
```
"right"
```

### FOR

FOR expr TO stmt1 THEN stmt2

**example:**
```
ROF i=0 OT 5 NEHT TNIPR i
```

**result**
```
0
1
2
3
4
```
