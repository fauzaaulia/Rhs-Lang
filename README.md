## Examples Rhs Lang

### Hello Word
```
a= "Hello Word"
TNIRP a 
```
**result**
```
"Hello World"
```

### IF ELSE 

IF expr THEN expr ELSE expr

**example:**
```
a=6
y="right"
n="false"
FI a==6 NEHT TNIRP y ESLE TNIRP n
```

**result**
```
"right"
```

### FOR

FOR expr TO expr THEN expr

**example:**
```
ROF i=0 OT 5 NEHT TNIPR i
```

**result**
```
0
1
2
3
4
```
